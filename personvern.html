2. Hva vi samler inn

Kontoopplysninger

  • Navn og e-postadresse
  • Kryptert passord — vi ser aldri ditt faktiske passord
  • Abonnementsplan og betalingsstatus
  • Tidspunkt for opprettelse og siste innlogging

Bruksdata

  • Lagrede søk og søkefiltre
  • Favorittmerkede objekter
  • Notater på objektsider (lagres kryptert)
  • IP-adresse og nettlesertype for sikkerhet og drift

Betalingsinformasjon

Kortinformasjon behandles av Stripe Inc. Vi lagrer aldri kortnummer, CVV eller utløpsdato — kun en anonymisert referanse og de siste fire sifrene for visningsformål.

Vi samler ikke inn fødselsnummer, helsedata, politisk ståsted eller andre særlige kategorier definert i GDPR art. 9.

4. Deling av data

Vi selger ikke personopplysninger. Vi deler kun med underleverandører som er nødvendige for driften:

  • Supabase Inc. — database og autentisering (EU, Frankfurt)
  • Stripe Inc. — betalingsbehandling
  • Resend / Postmark — transaksjonell e-post
  • Norske myndigheter — ved lovpålagt innsyn

Alle underleverandører er bundet av databehandleravtaler i samsvar med GDPR kapittel V.

6. Dine rettigheter

Under GDPR har du rett til innsyn, retting, sletting, begrensning, dataportabilitet og å trekke samtykke. Send forespørsler til personvern@eiendomsanalyse.no — vi svarer innen 30 dager.

Du kan slette kontoen din direkte fra Min side → Konto. Du kan også klage til Datatilsynet.

8. Sikkerhet

  • All datatrafikk krypteres med TLS 1.2+
  • Passord lagres som bcrypt-hash — vi kan aldri se dem
  • Database med row-level security i EU-region
  • Tilgangskontroll etter minste-privilegium-prinsippet

Ved sikkerhetsbrudd varsler vi Datatilsynet innen 72 timer og berørte brukere direkte ved høy risiko.

9. Endringer

Vesentlige endringer varsles på e-post minst 14 dager i forkant. Gjeldende versjon er alltid tilgjengelig på denne siden.
